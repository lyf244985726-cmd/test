# Avalon Online 设计

## 概述

基于浏览器的多人在线阿瓦隆桌游，使用点对点连接实现玩家之间的实时通信。

## 技术栈

- 前端：纯 HTML/CSS/JavaScript
- 联机：PeerJS (WebRTC)
- 部署：GitHub Pages

## 功能列表

### 房间系统
- 房主创建房间，生成唯一房间ID
- 生成邀请链接 `game.html?room=ID`
- 玩家点击链接自动加入
- 支持 5-10 人游戏

### 角色系统
- 好人阵营：梅林、派西维尔、亚瑟的忠臣
- 坏人阵营：莫德雷德、奥伯伦、刺客
- 根据玩家数量自动分配角色

### 游戏流程
1. 等待室 - 玩家加入、准备
2. 角色分配 - 私下显示角色
3. 任务阶段 - 队长选择队伍 → 投票 → 执行任务
4. 结果展示 - 任务成功/失败显示
5. 胜利判定 - 好人/坏人胜利

### 特殊规则
- 梅林知道谁是坏人（但不知道莫德雷德）
- 派西维尔知道梅林和莫甘娜
- 刺客任务 - 坏人可以刺杀梅林

## 部署

通过 GitHub Pages 托管，生成静态链接分享给朋友。

## 页面结构

- `index.html` - 首页（创建/加入房间）
- `game.html` - 游戏主页面
- `style.css` - 样式
- `game.js` - 游戏逻辑
- `peer.js` - P2P 连接逻辑
